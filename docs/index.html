⚔️ Character Sheet Scheda Personaggio

Think of it as a full digital character sheet that lives in your pocket. Create as many characters as you want, switch between them anytime, and every change is saved automatically — even if the bot restarts.

🧙 Create & Manage Characters

A guided wizard walks you through character creation: pick a name, choose one of the 12 D&D 5e classes (or type your own), set a subclass and starting level. You can create multiple characters on the same Telegram account and switch between them freely.

❤️ Hit Points & Armor Class

Track your health in real time. Set max HP, apply damage, heal, or set HP directly. Take a short rest to recover class resources, or a long rest to fully restore your health.

  • ⬆️ Set max HP
  • 📉 Apply damage
  • 💚 Heal
  • 🎯 Set current HP directly
  • 🌙 Short rest
  • ☀️ Long rest (full HP + resource reset)
  • 🛡️ Armor Class: base + shield + magic bonuses, calculated automatically

📊 Ability Scores & Skills

Set your six ability scores (STR, DEX, CON, INT, WIS, CHA) and the bot calculates modifiers automatically. All 18 D&D 5e skills are included — toggle proficiency, see your computed bonus, and roll skill checks with a single tap.

🔮 Spells & Spell Slots

A complete spell management system. Learn spells with a quick-add flow, edit every detail (casting time, range, components, duration, description), and cast them by consuming spell slots. The bot tracks concentration for you — only one concentration spell at a time, with automatic saving throw checks.

  • Fuzzy search — type a partial name and get instant results
  • Pin favourite spells to your character summary
  • Browse by level or as a flat list (configurable)
  • Manage spell slots independently — add, use, restore

🎒 Inventory

🗡️Weapon

AC value, stealth disadvantage
🔰Shield
AC bonus
🧪Consumable

🔧Tool
Tool type

Equip armor and shields to automatically update your AC. Equip multiple weapons at once. Adjust quantities on the fly.

💰 Currencies

Track all five D&D coin types — Copper, Silver, Electrum, Gold and Platinum — with add, subtract and automatic conversion.

✨ Special Abilities

Add passive or active abilities with usage limits and a restoration type (short rest, long rest, or manual). Spend and restore uses with a tap.

🔀 Multiclassing

Full multiclass support. Add additional classes, set subclasses, level up or down — class resources like Ki, Rage and Bardic Inspiration are recalculated automatically. Resting restores the right resources for each class.

🎲 Dice Roller

Roll any die from d4 to d100 with a configurable count (1–10). Results are saved in your roll history (last 50 rolls). A dedicated Initiative button rolls 1d20 + your DEX modifier with a single tap.

🌀 Conditions

📝 Notes & Voice Memos

🛡️ Live Group Party Party di Gruppo Live

Playing D&D with friends on Telegram? The party feature creates a live dashboard for your entire group — updated automatically every time something changes.

🚀 How It Works

1

Send /party in your Telegram group

2

Choose Public (visible to everyone) or Private (GM-only view)

3

A live status message appears showing every character’s state

👥 Who Appears?

Each player marks a character as “Party Active” in their settings. If someone only has one character, they’re included automatically. The bot tracks who writes in the group so it knows who’s in the session.

🔄 Real-Time Updates

The party message refreshes instantly whenever someone changes their HP, armor class, conditions or makes a dice roll. No manual refresh needed.

📋 What You See

For each character: name and class/level, a colour-coded HP bar, current AC, active conditions, and their last dice roll. The header shows the group name and remaining session time (sessions last 48 hours, or end early with /party_stop).

Giochi a D&D con gli amici su Telegram? La funzione party crea una dashboard live per tutto il gruppo — aggiornata automaticamente ogni volta che qualcosa cambia.

🚀 Come Funziona

1

Invia /party nel tuo gruppo Telegram

2

Scegli Pubblica (visibile a tutti) o Privata (solo per il GM)

3

Appare un messaggio di stato live che mostra lo stato di ogni personaggio

👥 Chi Appare?

Ogni giocatore segna un personaggio come “Party Attivo” nelle impostazioni. Se qualcuno ha un solo personaggio, viene incluso automaticamente. Il bot tiene traccia di chi scrive nel gruppo per sapere chi fa parte della sessione.

🔄 Aggiornamenti in Tempo Reale

Il messaggio party si aggiorna istantaneamente ogni volta che qualcuno modifica HP, classe armatura, condizioni o fa un tiro di dado. Nessun aggiornamento manuale necessario.

📋 Cosa Vedi

Per ogni personaggio: nome e classe/livello, una barra HP colorata, CA corrente, condizioni attive e l’ultimo tiro di dado. L’intestazione mostra il nome del gruppo e il tempo rimanente della sessione (le sessioni durano 48 ore, oppure si terminano prima con /party_stop).

/start

🚀 Quick Start Avvio Rapido

Want to run your own instance? You’ll need Python 3.10+ and a Telegram bot token from @BotFather.

1

Clone the repository and install dependencies

2

Create a .env file with your BOT_TOKEN

3

Run python -m bot.main and start playing!

💡 Tip: Add your Telegram chat ID as DEV_CHAT_ID in the .env to receive error reports directly in your DMs.
